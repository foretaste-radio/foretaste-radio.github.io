二集•解說：

第二集要開始播些比較屬於深層聆聽的東西了。

內容包括你在唱片中絕對聽不到的 keith jarrett 在八０年代初的獨奏會中間的朗誦詩和講話。日本庭園美學中的聲音裝置藝術：水琴窟。約翰凱基的公案。竹村延和的軟體電子音樂。中國女作曲家董魁的電腦程式 algorithmic 作曲。中島昭文的絕版「羊水」。加拿大的德國女性音景作曲家 hildegard westerkamp 的代表作 kits beach soundwalk，捷克的 artificial memory trace 的新具象音樂，等等。

hildegard westerkamp 這個作品極為重要，為什麼，因為它代表了整個加拿大式的音景作曲的關注。對四周對自然聲音的高度敏感，對環境聲音的人工調變。由作曲者自我旁白，引導聽者再度認識我們周遭的環境聲音，進一步以濾波器的頻率等化濾掉都市雜音，分離出自然聲音，再進一步將水生物的聲音與前衛音樂聲響作類比 （其中引用了 xenakis 的 Concret Ph）。凡是關心何為音景音樂創作的人必須聽這首作品。

什麼是真正電腦音樂？董魁是北京中央音樂院畢業、史丹佛大學音樂系作曲博士、現任 dartmouth college 作曲教授。她的這首 flying apples 是典型的 algorithmic composition 的範例。聲音演奏只是鋼琴聲，而真正用到電腦的是程式的自我邏輯生命。這也是外頭買不到的唱片。

捷克的 artificial memory trace 則代表了最尖端的第二甚至第三代的新具象音樂，完全不受老舊法國／加拿大式具象音樂的那一套可怕的理論、技術、美學教條的束縛。自由、隨心所欲。可聽性也高。

竹村延和 nobukazu takemura 則代表了當前最流行且重要的，用軟體，不是硬體，玩出來的電子音樂。

節目頭尾是用玄美好聽的 biosphere 和八木美知依包起來。^_^
