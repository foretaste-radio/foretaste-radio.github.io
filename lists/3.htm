三集•解說：

今天為何還要聽 LP ？

很多人錯認為只是音響玩家發燒友貪圖音響美色。非也！其實，對我們這種「音樂發燒友」來說，主要是為了音樂！！！你可知道有多少音樂從來就沒有發行過 CD 版？數量多得可怕！數十年來， LP 及 78 轉的唱片型式涵蓋了二十世紀人類的全部音樂出版。而轉成 CD 的只佔其中一小部份。更何況有極大比例的非主流 CD 也早已絕版。對只聽普普之音的樂友倒也無所謂，但你要是位音樂發燒友，還關心非主流音樂的話，透過當前市面上的 CD 能看到的現代音樂史就是非常局限偏狹的了。

至於唱片中何者會被選擇轉成 CD，我們會誤以為，重要的都轉了吧。其實不然啊。世界音樂史上極重要的 Shakti 第二第三張專集至今未全球發行 CD 版（僅有日本版）。連 Karlheinz Stockhausen 自己發行 CD 全集後，還是有沒轉成 CD 的重要作品漏網。

再拿大家以為比較熟悉的 ECM 公司來說，今天節目中介紹的這些 LP 唱片，每張都是五星級的重要作品，可是至今沒發行過 CD。就連 Keith Jarrett 在 ECM 也有一張唱片沒發行 CD，而他那套管風琴即興 Hymns/Spheres 也只發行了一半成為 CD。再拿已發行 CD 版的 Terje Rypdal 的 Odyssey 來說，CD 中省略了該 double album 專集中整個第四面的長達二十四分鐘的 "Rolling Stone" 一曲最重要作品。

你相信嗎？

這些音樂，在 ECM 的網站及目錄上甚至全部刪掉，隻字不提。比如，出過兩張精彩唱片的德國人 Ulrich Lask 你在網站樂人錄中根本找不到他。荒誕有如陰謀。樂友們可以到 http://www.trovar.com/ECMDiscography.html 樂迷網站看看完整的 ECM 出版目錄，那是歷史真相！

是，就是這麼回事，歷史就這樣地被改寫著。連國家不久前的歷史都能被自己篡改，音樂史又能算什麼？

不過，一如往常，我們會逆流而行…
